# Typy trojúhelníků

program na vstupu dostane číslo a vy vratíte "je kladne", "je záporná" nebo "je nula"

## Příklady
### Příklad 1
#### Vstup
```
10
```

#### Výstup
```
je kladne
```
---
### Příklad 2
#### Vstup
```
0
```

#### Výstup
```
je nula
```
---
### Příklad 3
#### Vstup
```
-103
```

#### Výstup
```
je záporná
```
